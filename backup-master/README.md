Os arquivos originais, exportações, dumps, mídia e cofres criptografados permanecem no Google Drive privado:

## Conteúdo permitido no GitHub

- documentação técnica sem credenciais;
- manifestos sem dados pessoais;
- scripts que não contenham segredos;
- checksums SHA-256;
- procedimentos de exportação e restauração;
- referências aos arquivos privados.

## Conteúdo proibido

Exportações do ChatGPT, dumps de banco, arquivos `.env`, chaves, tokens, seeds, códigos 2FA, credenciais, documentos pessoais, contratos privados, listas de clientes e mídias não autorizadas.

## Regra de conclusão

Um backup só é considerado concluído quando possuir data, origem, tamanho, SHA-256, versão ou commit e teste de restauração registrado.
